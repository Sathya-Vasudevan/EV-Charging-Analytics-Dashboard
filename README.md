##  Project Overview
This project analyzes Electric Vehicle (EV) charging data to understand energy consumption patterns, charging behavior, and station utilization.  
The goal is to help stakeholders optimize charging infrastructure and improve operational efficiency.

---

##  Objectives
- Analyze EV charging trends over time
- Identify high energy-consuming vehicle models
- Understand charging rate behavior under different conditions
- Measure charging station utilization
- Provide actionable insights using interactive dashboards

---

##  Tools & Technologies
- **Python**: Pandas, NumPy (Data Cleaning & Preprocessing)
- **Tableau**: Interactive Dashboard Creation
- **Jupyter Notebook**: Data Analysis
- **Excel/CSV**: Dataset Handling

---

##  Project Workflow
1. Raw data preprocessing using Python
2. Data cleaning and feature engineering
3. Exporting clean dataset for Tableau
4. Creating KPIs and visualizations
5. Dashboard design and insight generation

---

##  Key KPIs
- Total Energy Consumed
- Average Charging Duration
- Average Charging Rate
- Average SOC Increase
- Charging Station Utilization
- Peak Charging Hours
- Cost per kWh

---

##  Visualizations
- Energy Consumption Trend Over Time
- Top Vehicle Models by Energy Consumption
- Charging Rate vs Temperature 
- Charging Station Usage Analysis
- Day-of-Week and Time-of-Day Patterns
